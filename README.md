# lum.spreadsheet.php

## Summary

A library for quickly building Spreadsheet files using PhpSpreadsheet.

## Classes

| Class                   | Description                                       |
| ----------------------- | ------------------------------------------------- |
| Lum\Spreadsheet         | A wrapper around PhpOffice/PhpSpreadsheet.        |
| Lum\Worksheet           | Internal child class of Spreadsheet.              |
